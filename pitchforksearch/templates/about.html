This website will allow you to quickly search through Pitchfork review data and filter by a number of criteria. Pitchfork used to have this feature on their website but it was removed and I was bummed, so I built this tool for myself so I could quickly sort through old reviews, hopefully it can save you some time too!

Tools used in this application:

  • Flask Python based web frame
  • Mako Python templating engine
  • Flask-Mako Flask extension enabling Mako as the templating engine (I like it better than Jinja)
  • WTForms Python library for handling webforms
  • Flask-WTForms Flask extension for WTForms
  • Tablesorter Javascript library for the interactive table sorting (go figure)
  • Twitter Bootstrap Website layout and styles
